<h1 align="center">X-Spider</h1>

一个推特媒体（图片、视频）下载器。

可以批量添加博主名，批量一键下载、去重、设定下载范围、记录下载时间。


## 下载

[Releases](https://github.com/obihs2501/x-spider-mod-2026/releases/latest)

## 功能

- 媒体过滤器（如指定下载日期范围）
- 跳过已下载文件
- 可配置文件名、保存路径格式
- 手动、自动代理
- Cookie 登录
- **免登录下载**：未配置 Cookie 时自动激活「游客令牌」，无需登录即可下载公开账号的原画质图片与视频
- **单条帖子解析**：在搜索框直接粘贴帖子链接（如 `https://x.com/xxx/status/123`），仿 X 界面预览后一键下载
- **本地画廊**：按博主文件夹浏览已下载的图片和视频，支持调整每行个数与显示模式
- **增量下载**：自动记录已下载博主与上次下载时间，一键从上次位置继续下载
- **深浅色主题**：Claude 风格界面，设置中可切换浅色/深色模式
